## 1. SEO 전략 (Phase 1, 최우선)

### 1.1 즉시 필요

- [ ] **커스텀 도메인 구매** (.kr 또는 .com) — AdSense/SEO 전제조건 (**전 부서 1순위 합의**)
- [x] **meta description** — 정적 meta 추가 완료 (v0.9.0)
- [x] **OG tags** — og:title, og:description 추가 완료 (og:url은 도메인 확정 후)
- [x] **Twitter Card** tags — 추가 완료
- [x] **sitemap.xml** — 추가 완료 (상대 경로 → 도메인 확정 후 절대 URL 변경 필요)
- [x] **robots.txt** — 추가 완료
- [x] **canonical URL** — 추가 완료 (상대 경로 → 절대 URL 변경 필요)
- [ ] **구조화된 데이터** (JSON-LD, 선택) — 리치 스니펫
- [ ] **동적 OG 이미지** — SNS 공유 시 클릭률 결정 요소

### 1.2 검색엔진 등록

- [ ] **네이버 서치어드바이저** 등록 (한국 검색 60% 점유)
- [ ] **Google Search Console** 등록
- [ ] **Daum 검색등록** (선택)

### 1.3 SEO 핵심 이슈 (기술 연계)

| 문제 | 영향 | 해결 | 상태 |

**"한국의 모든 트렌드를 한눈에"** 또는 **"한국 인터넷의 실시간 온도계"**

- 네이버/다음: 뉴스 중심 → 우리: 뉴스 + 커뮤니티 + SNS + 정부 교차 분석
