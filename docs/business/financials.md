### 3.2 대체/보완 수익 모델

| 모델 | 예상 수익 | 전제조건 | 시점 |
|------|-----------|----------|------|
| **쿠팡 파트너스** | $100-500/월 | 도메인 + 가입 | P1 (즉시) |

| **B2B API** | 50-500만원/월 | DAU 1K+, 데이터 라이선스 | P4+ |
| **프리미엄 구독** | $5-10/유저/월 | 유저 계정, 차별화 기능 | P4+ |

### 3.3 수익 전략 우선순위

1. **AdSense + 쿠팡 파트너스** 동시 진행 (P1)
2. AdSense 거부 시 **카카오 AdFit** 즉시 전환 (P3)
3. **뉴스레터 광고** (구독자 500+ 확보 후, P3~P4)
4. **B2B API** (B2C 검증 후, 데이터 라이선스 확보 뒤)
