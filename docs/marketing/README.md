## 1. SEO 전략 (Phase 1, 최우선)

### 1.1 즉시 필요

- [ ] **커스텀 도메인 구매** (.kr 또는 .com) — AdSense/SEO 전제조건
- [ ] **meta description** — 각 카테고리 페이지별 동적 설명
- [ ] **OG tags** (og:title, og:description, og:image, og:url) — 소셜 공유 미리보기
- [ ] **Twitter Card** tags
- [ ] **sitemap.xml** 자동 생성 — 카테고리 페이지 기반
- [ ] **robots.txt** — 크롤러 가이드
- [ ] **canonical URL** — 중복 페이지 방지
- [ ] **구조화된 데이터** (JSON-LD, 선택) — 리치 스니펫

### 1.2 검색엔진 등록

- [ ] **Google Search Console** 등록

## 3. 리텐션 전략

### 3.1 현재 문제

- 사용자가 PostCard 클릭 → 외부 사이트로 이탈 → 재방문 동기 없음
- 푸시 알림, 이메일, 북마크 없음
- 방문 후 15-30초 내 이탈 (시간-on-site 극히 낮음)

### 3.2 해결 계획

| 메커니즘 | 시점 | 효과 |
|----------|------|------|
| **PWA "홈 화면에 추가"** | Phase 3 | 모바일 재방문률 2-3x |
| **KakaoTalk 일일 다이제스트** | Phase 3 | 습관 형성 (매일 오전 8시) |
| **이메일 뉴스레터** | Phase 4 | 리텐션 + 광고 수익 |
| **북마크/저장** 기능 | Phase 3 | 재방문 동기 |
| **읽음 표시** (localStorage) | Phase 3 | 재방문 시 새 글 쉽게 확인 |

## 6. 브랜드 포지셔닝

### 현재 포지션

"한국 커뮤니티 모아보기" (index.html title)

### 제안 포지션

**"한국의 모든 트렌드를 한눈에"** 또는 **"한국 인터넷의 실시간 온도계"**

- 네이버/다음: 뉴스 중심 → 우리: 뉴스 + 커뮤니티 + SNS + 정부 교차 분석
- 핵심 가치: "지금 한국에서 무엇이 뜨고 있는지" 30초 내 파악

### 타겟 유저 페르소나

| 페르소나 | 설명 | 니즈 |
|----------|------|------|
| 정보 중독자 | 다수 커뮤니티 + 뉴스 매일 확인 | 한 곳에서 모아보기 |
| 마케터/PR | 실시간 이슈 모니터링 | 크로스소스 트렌드 감지 |
| 해외 거주 한국인 | 한국 소식 놓치고 싶지 않음 | 일일 다이제스트 |
| 개발자/테크 관심자 | GeekNews + 커뮤니티 통합 | 테크 트렌드 큐레이션 |
