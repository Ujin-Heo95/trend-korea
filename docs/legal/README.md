## 핵심 법적 이슈

### 1. 웹 스크래핑 적법성

trend-korea는 28개 공개 소스에서 제목/URL/메타데이터를 수집합니다.

**현재 접근:**
- 본문 전체 복사 없음 — 제목과 링크만 수집
- robots.txt 준수 (차단된 경로 접근 안 함)
- 과도한 요청 방지 (p-limit 4, 10~30분 주기)
- 원본 링크 제공 (트래픽을 원본 사이트로 유도)

**법적 근거:**
- 저작권법 제28조 (인용) — 보도/비평/연구 목적의 인용 허용
- 그러나 저작권법 제93조 (데이터베이스 제작자 권리) — 체계적 추출은 침해 가능

**판례 참조:**
- 야놀자 v. 여기어때 (2023) — 체계적 자동 수집은 부정경쟁행위 가능
- 네이버 v. 사람인 라인 — 공개 데이터라도 대규모 자동 수집은 위법 가능

**리스크별 소스 분류:**

| 리스크 | 소스 유형 | 소스 | 조치 |
|--------|-----------|------|------|
| **낮음** | RSS 피드 | 뉴스 RSS 14개, 정부 RSS 3개, 기술 RSS 2개 등 | RSS는 재배포 목적 포맷 — 암묵적 허용 |
| **낮음** | 공식 API | YouTube Data API | ToS 준수 운영 |
| **중간** | HTML 스크래핑 | dcinside, bobaedream, ruliweb, theqoo, instiz, natepann, todayhumor | **ToS 감사 필요** |
| **높음** (제외됨) | — | 블라인드, 네이버 카페, 인스타그램 | 이미 제외 |

**필수 조치:**
- [ ] **P0**: 7개 HTML 스크래핑 사이트 ToS 개별 확인 (4시간)
- [ ] **P1**: 사이트 운영자에게 비공식 허락 이메일 발송 — "원문 링크 + 트래픽 유도" 취지 설명
- [ ] **P1**: C&D 수신 시 즉시 disable 프로토콜 문서화
- [ ] **P2**: "트렌드 리포팅" 편집 레이어 추가 → 공정 이용(fair use) 방어 강화

### 2. 개인정보 처리 (PIPA)

**현재 상태:**
- 사용자 회원가입/로그인 없음
- 쿠키, 개인정보 수집 없음

**크로스-패널 합의**: GA4 대신 Umami/Plausible (자체호스팅) 채택 → PIPA 이슈 최소화

**향후 분석도구 도입 시 (Phase 1):**
- [ ] **P0**: 개인정보 처리방침 페이지 작성 (4시간)
- [ ] **P1**: 쿠키 동의 배너 (세션 기반 반응 시스템 도입 시)
- [ ] 자체호스팅 분석도구 사용 시에도 IP 수집 여부 명시

**사용자 반응 시스템 도입 시 (Phase 3):**
- session_id는 `crypto.randomUUID()` 해시 — IP 주소 수집 안 함
- 개인정보처리방침에 세션 기반 데이터 수집 항목 추가

**국제 데이터 이전:**
- Railway US 서버 + 한국 사용자 = PIPA 제17조, 제39조의12 적용
- **해결**: Phase 4에서 아시아 리전 이전 (Fly.io Tokyo / Supabase Singapore)
- **임시 조치**: 개인정보처리방침에 데이터 처리 위치 명시

### 3. 사업자 등록 & 세무

- [ ] **P0**: 개인사업자등록 (AdSense 수익 발생 전 필수)
  - 업종: 정보서비스업 / 인터넷 포털
  - 간이과세자 가능 (연매출 8,000만원 이하)
- [ ] **P1**: 통신판매업 신고 (유료 서비스 도입 시)
- [ ] AdSense 수익 → 종합소득세 신고 (매년 5월)
- [ ] 해외 결제 수취 → 외화 수입 신고

### 4. 사이트 운영 표시 의무 (정보통신망법)

서비스 운영 시 다음 정보를 사이트에 표시해야 합니다:
- [ ] **P1**: 사업자 상호, 대표자명, 사업자등록번호
- [ ] **P1**: 연락처 (이메일, 전화번호)
- [ ] **P1**: 통신판매업 신고번호 (해당 시)

### 5. AdSense 컴플라이언스

**주의 사항:**
- AdSense는 "스크래핑된 콘텐츠" 사이트를 거부할 수 있음
- **대응**: 편집 레이어 (트렌드 스코어링, 크로스소스 분석, 일일 리포트) 추가로 "부가가치 어그리게이터"로 포지셔닝
- 자체 콘텐츠 비율 확보 (일일 리포트, 트렌드 분석)
