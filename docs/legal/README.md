# 법무

## 핵심 법적 이슈

### 1. 웹 스크래핑 적법성

trend-korea는 22개 공개 소스에서 제목/URL/메타데이터를 수집합니다.

**현재 접근:**
- 본문 전체 복사 없음 — 제목과 링크만 수집
- robots.txt 준수 (차단된 경로 접근 안 함)
- 과도한 요청 방지 (p-limit 4, 10~30분 주기)
- 원본 링크 제공 (트래픽을 원본 사이트로 유도)

**검토 필요 사항:**
- [ ] 각 소스별 이용약관 확인
- [ ] 저작권법 제28조 (인용) 적용 범위 확인
- [ ] 데이터베이스 제작자 권리 (저작권법 제93조) 검토
- [ ] RSS 피드의 경우 명시적 재배포 허용 여부 확인

### 2. 개인정보 처리

**현재 상태:**
- 사용자 회원가입/로그인 없음
- 쿠키, 개인정보 수집 없음
- GA4 연동 시 개인정보 처리방침 필요

**GA4 도입 시 필요 조치:**
- [ ] 개인정보 처리방침 페이지 작성
- [ ] 쿠키 동의 배너 (GDPR/개인정보보호법)
- [ ] GA4 익명화 설정

### 3. 사업자 등록

- [ ] 법인/개인사업자 설립 여부 결정
- [ ] AdSense 수익 발생 시 세금 신고 체계
