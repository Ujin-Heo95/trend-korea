# 경영기획

> 2026-03-29 비즈니스 패널 에이전트 분석 반영.

---

## 사업 개요

**Trend Korea**는 한국 주요 커뮤니티, 뉴스, YouTube에서 실시간 트렌딩 콘텐츠를 수집해
한 곳에서 보여주는 어그리게이터 서비스입니다.

**핵심 가치:** 여러 플랫폼을 돌아다니지 않아도 "지금 한국에서 뭐가 뜨는지" 한눈에 파악.

**핵심 차별화:** 뉴스 + 커뮤니티 + 정부 + 트렌드를 교차 분석하는 "크로스소스 트렌드 감지"

## 비전

공개된 실시간 정보 중 **가치가 높은 정보**를 큐레이션하여 제공하는 플랫폼.
단순 나열 → 인텔리전스(랭킹, 중복제거, 일일 리포트) → 개인화로 진화.
