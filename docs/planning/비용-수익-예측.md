## 4. 경쟁 환경

| 경쟁자 | 강점 | 약점 | 우리의 차별점 |
|--------|------|------|-------------|
| 네이버 뉴스 | 방대한 소스, 검색 연동 | 뉴스만, 커뮤니티 미포함 | 크로스소스 감지 |
| 다음 뉴스 | 카카오 생태계 | 뉴스만 | 커뮤니티 + 트렌드 통합 |
| Google News Korea | 글로벌 알고리즘 | 한국 커뮤니티 미포함 | 한국 특화 |
| 줌(Zum) | 포털 | 커뮤니티 약함 | 실시간 교차 분석 |

### 핵심 차별화 (moat)

**"한국 인터넷의 실시간 온도계"**
- 뉴스 + 커뮤니티 + SNS + 정부 발표를 교차 분석
- 동일 주제가 여러 소스에서 동시 등장 시 감지
- 네이버/다음은 뉴스만 클러스터링 → 커뮤니티 반응은 놓침
