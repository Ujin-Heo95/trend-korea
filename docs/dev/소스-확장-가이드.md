## 5. 소스 추가 절차 (기존 가이드 보완)

### RSS 소스 추가 (코드 0줄)

1. RSS 피드 URL 브라우저에서 직접 확인 (XML 응답 여부)
2. `sources.json`에 6줄 추가:
   ```json
   {
     "key": "source_key",
     "name": "소스 이름",
     "category": "카테고리",
     "type": "rss",
     "feedUrl": "https://...",
     "priority": "medium",
     "enabled": true
   }
   ```
3. 로컬에서 `npm run dev` 후 `/api/sources` 확인
4. `scraper_runs`에서 해당 소스 success 여부 확인
5. 7일 모니터링 → 0 posts 지속 시 disable

### HTML 스크래퍼 추가

1. [스크래퍼-추가-가이드.md](스크래퍼-추가-가이드.md) 참조
2. **추가 체크리스트**:
   - [ ] 대상 사이트 robots.txt 확인
   - [ ] 대상 사이트 ToS 스크래핑 조항 확인
   - [ ] `id`/`data-*` 속성 기반 셀렉터 우선 (class보다 안정적)
   - [ ] fetch() 내부에서 `catch { return [] }` 금지 → throw 해야 함

- 7일 연속 0 posts → disable 검토
