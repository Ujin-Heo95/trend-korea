# 스크래퍼 추가 가이드

## RSS 소스 추가 (코드 0줄)

`backend/src/scrapers/sources.json`에 다음 형태로 추가:

```json
{
  "key": "example",
  "name": "예시 뉴스",
  "category": "news",
  "type": "rss",
  "feedUrl": "https://example.com/rss.xml",
  "priority": "medium",
  "enabled": true
}
```

- `key`: 영문 소문자, 언더스코어 (DB source_key로 사용)
- `category`: community, news, tech, video, deals, alert 중 선택
- `priority`: high(10분), medium(15분), low(30분)
- `enabled`: false로 두면 수집 제외

추가 후 배포하면 자동으로 수집 시작.

## HTML 스크래퍼 추가 (1파일 + JSON)

### 1. 스크래퍼 파일 작성

`backend/src/scrapers/example.ts`:

```typescript
import axios from 'axios';
import * as cheerio from 'cheerio';
import type { Pool } from 'pg';
import { BaseScraper } from './base.js';
import type { ScrapedPost } from './types.js';

export class ExampleScraper extends BaseScraper {
  constructor(pool: Pool) { super(pool); }

  async fetch(): Promise<ScrapedPost[]> {
    const { data } = await axios.get('https://example.com/best', {
      headers: { 'User-Agent': 'Mozilla/5.0' },
    });
    const $ = cheerio.load(data);
    const posts: ScrapedPost[] = [];

    $('셀렉터').each((_, el) => {
      posts.push({
        sourceKey: 'example',
        sourceName: '예시',
        title: $(el).find('.title').text().trim(),
        url: $(el).find('a').attr('href') ?? '',
        category: 'community',
      });
    });

    return posts.slice(0, 30);
  }
}
```

### 2. sources.json에 등록

```json
{
  "key": "example",
  "name": "예시",
  "category": "community",
  "type": "html",
  "module": "./example.js",
  "className": "ExampleScraper",
  "priority": "high",
  "enabled": true
}
```

### 3. 테스트 작성

`backend/tests/scrapers/example.test.ts`에 fixture HTML + nock mock으로 테스트.

## API 소스 추가

HTML과 동일하나 `type: "api"` 사용. 외부 API 키가 필요하면 `config/index.ts`에 환경변수 추가.
