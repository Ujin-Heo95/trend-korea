# BD-001: 1차 수익 모델로 Google AdSense 선택

- 상태: 검토중
- 일자: 2026-03-29
- 부서: 경영기획

## 맥락

서비스 수익화 방안을 결정해야 함. 트래픽 기반 서비스로서 가능한 수익 모델:
1. 광고 (AdSense, 직접 광고 영업)
2. 프리미엄 구독 (알림, 필터, API)
3. 데이터 판매 (트렌드 분석 리포트)

## 결정

1차 수익 모델로 Google AdSense를 채택.

**이유:**
- 초기 트래픽이 적어도 시작 가능 (최소 요건: 콘텐츠 충분, 정책 준수)
- 구현 비용이 가장 낮음 (ads.txt + 광고 컴포넌트)
- 구독 모델은 충분한 사용자 기반이 없으면 전환율이 극히 낮음
- 데이터 판매는 분석 파이프라인 구축 비용이 높음

## 대안

| 대안 | 미선택 이유 |
|------|------------|
| 직접 광고 영업 | 영업 인력 필요, 초기 단계에 비현실적 |
| 프리미엄 구독 | 사용자 기반 부족, 무료 대안 많음 |
| 데이터 API 판매 | 분석 인프라 필요, 수요 미검증 |

## 결과

- AdSense 승인을 위해 프론트엔드 SEO 작업 선행 필요
- 도메인 구매 + 콘텐츠 품질 기준 충족 필요
- 향후 트래픽 확보 후 프리미엄 구독 모델 추가 검토
